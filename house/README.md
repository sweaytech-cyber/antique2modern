# Luxury Housewarming Invitation Website - 14220 W North Oak Blvd

This repository contains a premium, mobile-first, interactive housewarming invitation website for **14220 W North Oak Blvd**.

## ✨ Features
- **3D Room Carousel**: Auto-rotating and touch-swipeable 3D viewport showcasing high-quality images of the house.
- **Midnight Gold Design**: Luxury styling using glassmorphism, responsive CSS layout, and elegant serif typography.
- **Interactive RSVP Wax Seal**: RSVP submission flips to reveal a sealed envelope stamped with an animated monogram wax seal.
- **Embedded Audio**: Relaxing ambient lounge music track with play/pause controls.
- **Host Dashboard Log**: Access the RSVP database with the PIN code `14220` (entered in the footer button) and export submissions directly to CSV.

## 🚀 Deployment Instructions for GitHub Pages
You can host this website for free on GitHub Pages in under 2 minutes:
1. Create a new repository on GitHub (e.g., `housewarming-invite`).
2. Copy and paste all the files in this folder (`index.html`, `README.md`, and the `assets` folder) into the repository.
3. Commit and push the files to the `main` branch.
4. Go to **Settings** -> **Pages** in your GitHub repository.
5. Under **Build and deployment**, select **Deploy from a branch**.
6. Set the branch to `main` (and folder `/ (root)`), then click **Save**.
7. In a few seconds, GitHub will provide a public link (e.g., `https://username.github.io/housewarming-invite/`) to share with guests!

## ⚙️ Customization
- **Change Details**: To change the date, time, location, or other details, open `index.html` and edit the text content inside the HTML layout or the settings inside the `<script>` block.
- **Host PIN**: Modify the PIN in `index.html` (currently set to `14220`) for secure host access.
